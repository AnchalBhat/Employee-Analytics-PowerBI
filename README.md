# Workforce Analytics Dashboard 📊

## Project Overview
This project focuses on analyzing workforce data to generate meaningful insights for HR decision-making. Using structured employee data, the system helps in understanding workforce distribution, performance trends, and key HR metrics.

It is designed to support data-driven HR strategies and improve organizational efficiency.

---

## Objectives
- Analyze employee data for better workforce planning  
- Identify performance trends and patterns  
- Improve decision-making using data insights  
- Automate data collection and reporting  

---

## Tools & Technologies Used
- Power BI (for data visualization)  
- Google Forms (for data collection)  
- Google Sheets (for data storage & processing)  
- Microsoft Excel / CSV Dataset  

---

## Dataset Information
The dataset includes employee-related information such as:
- Employee demographics  
- Department details  
- Performance metrics  
- Workforce distribution  

*(Dataset used: `powerbi_ready_dataset.csv`)*

---

## Key Features
- Interactive dashboards for HR insights  
- Workforce distribution analysis  
- Performance tracking  
- Automated data collection system  
- Easy-to-understand visual reports  

---

## How It Works
1. Data is collected using Google Forms  
2. Responses are stored in Google Sheets  
3. Dataset is processed and cleaned  
4. Power BI dashboard is created for visualization  
5. Insights are generated for HR decision-making
